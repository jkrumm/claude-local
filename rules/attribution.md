# No Attribution

Never add AI or tool attribution anywhere:

- **Code comments**: No "Generated by AI", "Claude wrote this"
- **Commit messages**: No `Co-Authored-By: Claude...` footer
- **PR descriptions**: No "Created with Claude Code" footer, no "Generated with AI" disclaimers, no "CodeRabbit" or other review tool mentions
- **Documentation**: No "AI-generated" disclaimers
- **PR/commit footers**: No mentions of CodeRabbit, SonarQube, Cursor, Copilot, or any other AI/tooling that reviewed or generated the code

AI assistance is tooling, not authorship. Keep all artifacts clean and professional.
